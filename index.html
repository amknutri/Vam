Como funciona o cashback?

1. Empresário oferece desconto

Você define o percentual (ex: 10%). Esse valor vai para a carteira MRP Mobi.

2. Cliente ganha cashback imediato

30% do valor do desconto volta para o cliente na hora.

3. O restante é distribuído em rede

45% do valor retido pela MRP é dividido entre os indicadores (até o 7º nível). Os outros 55% são da MRP.
